## 한국어

### 개요
**Hide morning in Calendar** (이하 "본 확장")은 Google Calendar 주간 보기에서 아침 시간대를 숨기는 Chrome 확장 프로그램입니다. 본 확장은 사용자의 **개인정보를 수집하지 않으며 외부로 전송하지 않습니다.**

### 수집·전송하는 데이터
- **수집하는 개인정보**: 없음
- **외부 서버로 전송하는 데이터**: 없음
- **분석/트래킹 도구**: 사용하지 않음
- **광고 SDK**: 포함되지 않음
- **쿠키**: 사용하지 않음

### 로컬에 저장되는 항목
다음 설정값은 사용자의 브라우저 내 `chrome.storage.local`에만 저장됩니다. 본 확장 외에는 어떤 주체도 이 값을 읽거나 전송하지 않습니다.

- 하루 시작 시간 (start time)
- 수면 시간 (sleep duration)
- Bedtime Coach 활성화 여부

해당 데이터는 확장 프로그램을 제거하면 함께 삭제됩니다.

### 권한 사용 목적
- **`storage`**: 위 설정값을 사용자 브라우저에 로컬 저장하기 위해 사용합니다.
- **`tabs`**: 열려있는 여러 Google Calendar 탭 사이에 설정 변경을 동기화하기 위해 사용합니다. 탭의 콘텐츠를 읽거나 외부로 전송하지 않습니다.
- **Host permission (`https://calendar.google.com/calendar/*`)**: Google Calendar 페이지에서만 콘텐츠 스크립트가 실행되도록 제한합니다. 그 외 어떤 사이트에서도 동작하지 않습니다.

### 제3자 데이터 공유
본 확장은 어떠한 제3자(서버, 분석 업체, 광고주 등)와도 데이터를 공유하지 않습니다.

### 아동의 개인정보
본 확장은 13세 미만 아동을 대상으로 하지 않으며, 아동으로부터 개인정보를 수집하지 않습니다.

### 정책 변경
본 정책이 변경될 경우 본 페이지 상단의 "Last updated" 날짜가 갱신됩니다. 중요한 변경 사항은 확장 프로그램 업데이트 노트에 명시됩니다.

### 문의
개인정보 관련 문의는 본 확장의 GitHub 저장소 Issues 또는 Chrome Web Store 개발자 연락처로 남겨주세요.

## English

### Overview
**Hide morning in Calendar** (the "Extension") is a Chrome extension that hides early-morning hours in the Google Calendar weekly view. The Extension **does not collect any personal data, and does not transmit any data to external servers.**

### Data we collect or transmit
- **Personal information collected**: None
- **Data sent to external servers**: None
- **Analytics / tracking**: None
- **Advertising SDKs**: None
- **Cookies**: None

### Data stored locally
The following user preferences are stored only inside the user's browser via `chrome.storage.local`. No party other than the Extension itself reads or transmits these values.

- Day start time
- Sleep duration
- Bedtime Coach enabled / disabled

This data is removed when the user uninstalls the Extension.

### Permissions usage
- **`storage`** — stores the preferences listed above, locally in the user's browser.
- **`tabs`** — synchronizes setting changes across open Google Calendar tabs. Tab contents are neither read nor transmitted.
- **Host permission for `https://calendar.google.com/calendar/*`** — restricts the content script to Google Calendar pages only. The Extension does not run on any other site.

### Third-party data sharing
The Extension does not share any data with any third party (servers, analytics providers, advertisers, etc.).

### Children's privacy
The Extension is not directed at children under 13 and does not knowingly collect personal information from children.

### Changes to this policy
If this policy changes, the "Last updated" date at the top of this page will be revised. Material changes will also be noted in the Extension's release notes.

### Contact
For privacy-related inquiries, please open an issue on the GitHub repository or use the developer contact listed on the Chrome Web Store page.
